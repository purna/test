# test

A test repository wired to [pixelKanban](https://github.com/purna/pixelKanban) — a lightweight, GitHub-Issues–backed Kanban board deployed via GitHub Pages.

## Setup

1. Trigger the **Deploy pixelKanban to GitHub Pages** workflow from the **Actions** tab (workflow dispatch).
2. Once the workflow completes, enable GitHub Pages in **Settings → Pages**, setting the source branch to `gh-pages`.
3. The Kanban board will be available at `https://<owner>.github.io/<repo>/` and will display issues from this repository.
